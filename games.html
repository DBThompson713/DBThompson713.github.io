1 Bitcoin

Ruby terminal game

A game that I designed to be as confusing as the bitcoin market.

1 Bitcoin features 8-bit graphics and makes the user guess the number of common australian items purchasable with 1 Bitcoin. The value of 1 bitcoin is retaken live on every guess, which potentially changes the correct answer for each guess.
